# English Dictionary App

A simple English Dictionary web application built using HTML, CSS, and JavaScript. It allows users to search for English words and view their meanings along with pronunciation audio using a free dictionary API.

## Features

* Search for any English word
* Display word meaning
* Play pronunciation audio (if available)
* Responsive and user-friendly interface
* Handles invalid word searches

## Technologies Used

* HTML
* CSS
* JavaScript
* Dictionary API

## Project Structure

```
├── index.html
├── style.css
├── index.js
└── README.md
```

## How to Use

1. Open the application in your browser.
2. Enter a word in the search box.
3. Press **Enter**.
4. View the word's meaning and pronunciation.
